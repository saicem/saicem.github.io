# TeX

## 安装宏包

1. 放入宏包目录下 `\texmf-local\tex\latex`
2. 运行命令 `texhash` 更新目录以便Latex发行版能找到新安装的宏包

## 查询文档

- 命令行输入 `texdoc 宏包名`
- [从网站查询](https://texdoc.org/index.html)
