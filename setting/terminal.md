## Terminal 常用快捷键
{% note blue %}
- `alt+shift+plus` 新垂直窗格
- `alt+shift+-` 新水平窗格
- 按住`alt`键即可切换窗格
- 按住`alt+shift`并使用箭头键调整焦点窗格的大小
- 键入`ctrl+shift+w`来关闭焦点窗格
{% endnote %}

{% note blue %}
- `ctrl+shift+t` New tab
- `ctrl+tab` Open next tab
- `ctrl+shift+tab` Open previous tab
{% endnote %}
